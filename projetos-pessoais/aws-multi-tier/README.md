# ☁️ Infraestrutura Cloud Multi-Tier (AWS)

Documentação e scripts referentes à "Fase 4" do projeto de infraestrutura em nuvem. Este projeto demonstra a capacidade de planejar e implementar uma arquitetura de alta disponibilidade, dividida em múltiplas camadas.

### 🛠️ Serviços AWS Utilizados
* **VPC (Virtual Private Cloud):** Isolamento de rede e configuração de sub-redes públicas e privadas.
* **EC2 Auto Scaling:** Instâncias de computação elásticas para garantir performance e disponibilidade.
* **Amazon RDS:** Banco de dados relacional gerenciado e isolado na camada privada.

### 🎯 Objetivo
Projetar um ambiente seguro e escalável, aplicando conceitos de arquitetura corporativa para suportar aplicações web robustas, separando as camadas de apresentação (web) e de dados.
